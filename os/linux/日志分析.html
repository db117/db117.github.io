# grep

  • 在文件demo.log中查找所有包行INFO的行
    • grep 'INFO' demo.log
    • -a 7 后七行
    • -b 7 前七行
  • 输出文件demo.log中查找所有包行ERROR的行的数量
    • grep -c 'ERROR' demo.log
  • 查找不含"ERROR"的行
    • grep -v 'ERROR' demo.log
