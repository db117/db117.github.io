小技巧 ​

向上取整 ​

(m + (n - 1)) / n; 
相当于
int i=m/n;
if(m%n!=0){
	i++;
}

1 累加到 n ​

因为S=1+2+3+...+n，并且，S=n+(n-1)+(n-2)+...+1，把这两个等式左右分别相加可以得到：2S=(n+1)+(n+1)+(n+1)+...+(n+1)，其中等式右边一共是n个（n+1)相加是很容易数出来的，所以得到 2S=n(n+1)，于是S=n(n+1)/2

n * (n + 1) / 2
