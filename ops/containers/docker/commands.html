run вҖӢ

docker run [OPTIONS] IMAGE [COMMAND] [ARG...]

  • OPTIONSиҜҙжҳҺпјҡ
    • -a stdin: жҢҮе®ҡж ҮеҮҶиҫ“е…Ҙиҫ“еҮәеҶ…е®№зұ»еһӢпјҢеҸҜйҖү STDIN/STDOUT/STDERR дёүйЎ№пјӣ
    • -d: еҗҺеҸ°иҝҗиЎҢе®№еҷЁпјҢе№¶иҝ”еӣһе®№еҷЁIDпјӣ
    • -i: д»ҘдәӨдә’жЁЎејҸиҝҗиЎҢе®№еҷЁпјҢйҖҡеёёдёҺ -t еҗҢж—¶дҪҝз”Ёпјӣ
    • -p: з«ҜеҸЈжҳ е°„пјҢж јејҸдёәпјҡдё»жңә(е®ҝдё»)з«ҜеҸЈ:е®№еҷЁз«ҜеҸЈ
    • -t: дёәе®№еҷЁйҮҚж–°еҲҶй…ҚдёҖдёӘдјӘиҫ“е…Ҙз»Ҳз«ҜпјҢйҖҡеёёдёҺ -i еҗҢж—¶дҪҝз”Ёпјӣ
    • --name="nginx-lb": дёәе®№еҷЁжҢҮе®ҡдёҖдёӘеҗҚз§°пјӣ
    • --add-host:ж·»еҠ еҹҹеҗҚipжҳ е°„
    • --dns 8.8.8.8: жҢҮе®ҡе®№еҷЁдҪҝз”Ёзҡ„DNSжңҚеҠЎеҷЁпјҢй»ҳи®Өе’Ңе®ҝдё»дёҖиҮҙпјӣ
    • --dns-search example.com: жҢҮе®ҡе®№еҷЁDNSжҗңзҙўеҹҹеҗҚпјҢй»ҳи®Өе’Ңе®ҝдё»дёҖиҮҙпјӣ
    • -h "mars": жҢҮе®ҡе®№еҷЁзҡ„hostnameпјӣ
    • -e username="ritchie": и®ҫзҪ®зҺҜеўғеҸҳйҮҸпјӣ
    • --env-file=[]: д»ҺжҢҮе®ҡж–Үд»¶иҜ»е…ҘзҺҜеўғеҸҳйҮҸпјӣ
    • --cpuset="0-2" or --cpuset="0,1,2": з»‘е®ҡе®№еҷЁеҲ°жҢҮе®ҡCPUиҝҗиЎҢпјӣ
    • -m :и®ҫзҪ®е®№еҷЁдҪҝз”ЁеҶ…еӯҳжңҖеӨ§еҖјпјӣ
    • --net="bridge": жҢҮе®ҡе®№еҷЁзҡ„зҪ‘з»ңиҝһжҺҘзұ»еһӢпјҢж”ҜжҢҒ bridge/host/none/container: еӣӣз§Қзұ»еһӢпјӣ
    • --link=[]: ж·»еҠ й“ҫжҺҘеҲ°еҸҰдёҖдёӘе®№еҷЁпјӣ
    • --expose=[]: ејҖж”ҫдёҖдёӘз«ҜеҸЈжҲ–дёҖз»„з«ҜеҸЈпјӣ

rm еҲ йҷӨдёҖдёӘжҲ–еӨҡе°‘е®№еҷЁ вҖӢ

docker rm [OPTIONS] CONTAINER [CONTAINER...]

  • OPTIONSиҜҙжҳҺпјҡ

    -f :йҖҡиҝҮSIGKILLдҝЎеҸ·ејәеҲ¶еҲ йҷӨдёҖдёӘиҝҗиЎҢдёӯзҡ„е®№еҷЁ

    -l :з§»йҷӨе®№еҷЁй—ҙзҡ„зҪ‘з»ңиҝһжҺҘпјҢиҖҢйқһе®№еҷЁжң¬иә«

    -v :-v еҲ йҷӨдёҺе®№еҷЁе…іиҒ”зҡ„еҚ·

exec пјҡеңЁиҝҗиЎҢзҡ„е®№еҷЁдёӯжү§иЎҢе‘Ҫд»Ө вҖӢ

docker exec [OPTIONS] CONTAINER COMMAND [ARG...]

  • OPTIONSиҜҙжҳҺпјҡ

    -d :еҲҶзҰ»жЁЎејҸ: еңЁеҗҺеҸ°иҝҗиЎҢ

    -i :еҚідҪҝжІЎжңүйҷ„еҠ д№ҹдҝқжҢҒSTDIN жү“ејҖ

    -t :еҲҶй…ҚдёҖдёӘдјӘз»Ҳз«Ҝ

ps : еҲ—еҮәе®№еҷЁ вҖӢ

docker ps [OPTIONS]

  • OPTIONSиҜҙжҳҺпјҡ

    -a :жҳҫзӨәжүҖжңүзҡ„е®№еҷЁпјҢеҢ…жӢ¬жңӘиҝҗиЎҢзҡ„гҖӮ

    -f :ж №жҚ®жқЎд»¶иҝҮж»ӨжҳҫзӨәзҡ„еҶ…е®№гҖӮ

    --format :жҢҮе®ҡиҝ”еӣһеҖјзҡ„жЁЎжқҝж–Үд»¶гҖӮ

    -l :жҳҫзӨәжңҖиҝ‘еҲӣе»әзҡ„е®№еҷЁгҖӮ

    -n :еҲ—еҮәжңҖиҝ‘еҲӣе»әзҡ„nдёӘе®№еҷЁгҖӮ

    --no-trunc :дёҚжҲӘж–ӯиҫ“еҮәгҖӮ

    -q :йқҷй»ҳжЁЎејҸпјҢеҸӘжҳҫзӨәе®№еҷЁзј–еҸ·гҖӮ

    -s :жҳҫзӨәжҖ»зҡ„ж–Үд»¶еӨ§е°ҸгҖӮ

cp :з”ЁдәҺе®№еҷЁдёҺдё»жңәд№Ӣй—ҙзҡ„ж•°жҚ®жӢ·иҙқгҖӮ вҖӢ

```docker cp [OPTIONS] CONTAINER:SRC_PATH DEST_PATH|-```
```docker cp [OPTIONS] SRC_PATH|- CONTAINER:DEST_PATH ```

images : еҲ—еҮәжң¬ең°й•ңеғҸгҖӮ вҖӢ

```docker images [OPTIONS] [REPOSITORY[:TAG]]```

  • OPTIONSиҜҙжҳҺпјҡ

    -a :еҲ—еҮәжң¬ең°жүҖжңүзҡ„й•ңеғҸпјҲеҗ«дёӯй—ҙжҳ еғҸеұӮпјҢй»ҳи®Өжғ…еҶөдёӢпјҢиҝҮж»ӨжҺүдёӯй—ҙжҳ еғҸеұӮпјүпјӣ

    --digests :жҳҫзӨәй•ңеғҸзҡ„ж‘ҳиҰҒдҝЎжҒҜпјӣ

    -f :жҳҫзӨәж»Ўи¶іжқЎд»¶зҡ„й•ңеғҸпјӣ

    --format :жҢҮе®ҡиҝ”еӣһеҖјзҡ„жЁЎжқҝж–Үд»¶пјӣ

    --no-trunc :жҳҫзӨәе®Ңж•ҙзҡ„й•ңеғҸдҝЎжҒҜпјӣ

    -q :еҸӘжҳҫзӨәй•ңеғҸIDгҖӮ

rmi : еҲ йҷӨжң¬ең°дёҖдёӘжҲ–еӨҡе°‘й•ңеғҸгҖӮ вҖӢ

  • docker rmi [OPTIONS] IMAGE [IMAGE...]

  • OPTIONSиҜҙжҳҺпјҡ

    -f :ејәеҲ¶еҲ йҷӨпјӣ

    --no-prune :дёҚз§»йҷӨиҜҘй•ңеғҸзҡ„иҝҮзЁӢй•ңеғҸпјҢй»ҳи®Өз§»йҷӨпјӣ

build е‘Ҫд»Өз”ЁдәҺдҪҝз”Ё Dockerfile еҲӣе»әй•ңеғҸгҖӮ вҖӢ

  • docker build [OPTIONS] PATH | URL | -

  • OPTIONSиҜҙжҳҺпјҡ

    --build-arg=[] :и®ҫзҪ®й•ңеғҸеҲӣе»әж—¶зҡ„еҸҳйҮҸпјӣ

    --cpu-shares :и®ҫзҪ® cpu дҪҝз”ЁжқғйҮҚпјӣ

    --cpu-period :йҷҗеҲ¶ CPU CFSе‘Ёжңҹпјӣ

    --cpu-quota :йҷҗеҲ¶ CPU CFSй…Қйўқпјӣ

    --cpuset-cpus :жҢҮе®ҡдҪҝз”Ёзҡ„CPU idпјӣ

    --cpuset-mems :жҢҮе®ҡдҪҝз”Ёзҡ„еҶ…еӯҳ idпјӣ

    --disable-content-trust :еҝҪз•Ҙж ЎйӘҢпјҢй»ҳи®ӨејҖеҗҜпјӣ

    -f :жҢҮе®ҡиҰҒдҪҝз”Ёзҡ„Dockerfileи·Ҝеҫ„пјӣ

    --force-rm :и®ҫзҪ®й•ңеғҸиҝҮзЁӢдёӯеҲ йҷӨдёӯй—ҙе®№еҷЁпјӣ

    --isolation :дҪҝз”Ёе®№еҷЁйҡ”зҰ»жҠҖжңҜпјӣ

    --label=[] :и®ҫзҪ®й•ңеғҸдҪҝз”Ёзҡ„е…ғж•°жҚ®пјӣ

    -m :и®ҫзҪ®еҶ…еӯҳжңҖеӨ§еҖјпјӣ

    --memory-swap :и®ҫзҪ®Swapзҡ„жңҖеӨ§еҖјдёәеҶ…еӯҳ+swapпјҢ"-1"иЎЁзӨәдёҚйҷҗswapпјӣ

    --no-cache :еҲӣе»әй•ңеғҸзҡ„иҝҮзЁӢдёҚдҪҝз”Ёзј“еӯҳпјӣ

    --pull :е°қиҜ•еҺ»жӣҙж–°й•ңеғҸзҡ„ж–°зүҲжң¬пјӣ

    --quiet, -q :е®үйқҷжЁЎејҸпјҢжҲҗеҠҹеҗҺеҸӘиҫ“еҮәй•ңеғҸ IDпјӣ

    --rm :и®ҫзҪ®й•ңеғҸжҲҗеҠҹеҗҺеҲ йҷӨдёӯй—ҙе®№еҷЁпјӣ

    --shm-size :и®ҫзҪ®/dev/shmзҡ„еӨ§е°ҸпјҢй»ҳи®ӨеҖјжҳҜ64Mпјӣ

    --ulimit :Ulimitй…ҚзҪ®гҖӮ

    --tag, -t: й•ңеғҸзҡ„еҗҚеӯ—еҸҠж ҮзӯҫпјҢйҖҡеёё name:tag жҲ–иҖ… name ж јејҸпјӣеҸҜд»ҘеңЁдёҖж¬Ўжһ„е»әдёӯдёәдёҖдёӘй•ңеғҸи®ҫзҪ®еӨҡдёӘж ҮзӯҫгҖӮ

    --network: й»ҳи®Ө defaultгҖӮеңЁжһ„е»әжңҹй—ҙи®ҫзҪ®RUNжҢҮд»Өзҡ„зҪ‘з»ңжЁЎејҸ
