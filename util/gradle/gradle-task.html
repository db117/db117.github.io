# JavaExec

JavaExec (opens new window)

就是执行java class的main方法

# 简单使用

plugins {
    id 'java'
}

task runApp(type: JavaExec) {
  classpath = sourceSets.main.runtimeClasspath
  mainClass = 'package.Main'
  // main参数
  args 'appArg1'
}

// 创建jar
jar {
  manifest {
    attributes('Main-Class': 'package.Main')
  }
}

task runExecutableJar(type: JavaExec) {
  // 只能有一个jar存在
  classpath = files(tasks.jar)

  // main可以不定义

  // main参数
  args 'appArg1'
}

# 属性

属性 描述
allJvmArgs (opens new window) 用于启动进程的 JVM 的完整参数集
args (opens new window) main方法参数
classpath (opens new window) classpath
enableAssertions (opens new window) 是否启用断言
environment (opens new window) 环境变量
errorOutput (opens new window) 错误输出. 默认 System.err.
mainClass (opens new window) 需要执行的类
maxHeapSize (opens new window) 最大堆大小
systemProperties (opens new window) 系统属性
workingDir (opens new window) 工作目录，默认项目目录
